Components > The Box

The complete banner to animations, ads and much more...

  • Create the correct Structure
  • Call jQuery function: $('.thebox').thebox();
Example:

                  $('.the-menu').themenu();
Parameters
Option Values Default value Description
Option Values Default value Description
speed Number 500(miliseconds) Set the speed of image/slide transition.
timer Number 400(miliseconds) Inicial time animation
showButtons Boolean false Show or not, the buttons left and right
bgOpacity Number 0.6 Opacity of BG
zoom100pct Boolean false Set if the image can be resized/initial width/height 100% (proporcional)
base String "body" Set the base to insert thebox.
mini Boolean false See miniatures in the Gallery.
Data Attributes (in element)
data-title String "" Add a Title Box (h3)
data-iframe String "" If you need use an iframe, set the url (src) in this option.
data-iframe-width Number or Pct "" Iframe Width
data-iframe-height Number or Pct "" Iframe Height
Data Attributes (in div target *href*)
data-width String "" Set width of window.
data-height String "" Set height of window.
Callbacks
afterClose Function "" Set a function after close the box. ex: {afterClose(functionname()};
